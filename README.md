# BRIK

Desarrollo una **plataforma SaaS de gestión integral para ferreterías, corralones y depósitos**, que permita:

- Vender en mostrador de forma rápida
- Ordenar consultas y presupuestos
- Controlar stock y entregas
- Gestionar clientes, cuentas corrientes y deudas
- Tener visibilidad clara del negocio sin complejidad técnica

Todo desde **un único sistema web**, multi-tenant, simple de usar, económico y sin necesidad de configuraciones complejas ni consultoría.

> El sistema impone un flujo claro del rubro, adaptable a negocios chicos, medianos y grandes, sin personalizaciones por cliente.
